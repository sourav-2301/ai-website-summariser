# AI Website Summariser

A simple React web application that takes a public website URL, fetches its content, and generates a short summary using the Groq AI API.

---

## Features

- Input field to enter any public URL
- Fetches website text using a CORS-friendly method
- Summarises website content using Groq AI
- Loading spinner while summarising
- Clean and responsive UI
  

---

## Tech Stack

- **Frontend:** React, HTML, CSS
- **AI API:** Groq AI
- **Other:** Vite, JavaScript
